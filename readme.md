# is18(_)
Is **`this`** 18?

## Install
### Through terminal
```sh
$ .. install purpleblueslime/is18
```
## Import
```js
import {is18} from 'is18';
```

## is18(18)
```js
is18(18) // YES
```

## is18(':3')
```js
is18(':3') // NO
```
